# Walmart Demand Forecasting & Inventory Optimization

### 🚀 The Business Problem
Retailers lose millions when they overstock (money wasted on storage) or understock (lost sales). This project predicts future demand and suggests exact inventory levels.

### 🛠️ The Tech Stack
- **Python** (Pandas/NumPy)
- **Prophet** (Time Series Forecasting by Meta)
- **Matplotlib/Seaborn** (Visualizing trends)

### 📈 Key Insights Found
- **Weekly Seasonality:** Sales spike significantly on weekends.
- **Holiday Impact:** High correlation between holiday flags and 25% revenue growth.
- **Optimization:** Implemented a 'Safety Stock' logic to ensure 98% service levels.
